# 🧠 VisionClassify AI

An AI-powered Image Classification web application built using **Flask**, **TensorFlow**, and **MobileNetV2**. Upload an image and let the AI identify the object with confidence scores and top predictions.

---

## ✨ Features

- 📷 Upload Images
- 🖼️ Live Image Preview
- 🤖 AI Image Classification
- 📊 Confidence Score
- 🏆 Top 3 Predictions
- ⚡ MobileNetV2 (ImageNet)
- 🎨 Modern Responsive UI
- 📱 Mobile Friendly
- 🖱️ Drag & Drop Upload
- 🔥 Loading Animation
- 🚀 Flask Backend

---

### Frontend

- TensorFlow

- ImageNet

---

## 🧠 AI Model

This project uses **MobileNetV2**, a lightweight Convolutional Neural Network pretrained on the **ImageNet** dataset containing over **1,000 object categories**.
